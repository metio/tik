# incident-response: declared

An incident exists once its severity and impact are on record.

- `tik set <id> severity=:sev1|:sev2|:sev3`
- `tik set <id> impact=who is affected and how`

Declare early and cheap — severity can be corrected later, and the
correction (like everything) is part of the record.
