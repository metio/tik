# Runbook: tik-dev / :triaged

Assert `summary` (what and why, one breath) and `kind`
(feature/bug/docs/spike) as a maintainer. If it duplicates existing
work, link it (`link.duplicate-of=<uuid>`) instead of triaging twice.
If it should wait, this is also where you park it — with the reason,
not silently.
