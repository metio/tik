# track: open

The ticket exists. There is no workflow to satisfy — record what is
true as it happens:

- `tik set <id> key=value` for facts worth keeping.
- `tik comment <id> <text>` for anything else.
- `tik attach <id> <file>` for evidence.

When several track tickets start following the same informal shape,
run `tik author` and give that shape a real definition.
